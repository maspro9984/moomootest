# moomoo 日本株リアルタイム相場 テストツール（Web版）

moomoo 証券の OpenAPI（`moomoo-api` / OpenD ゲートウェイ）を使って、
日本株のリアルタイム相場（例: 9984 ソフトバンクグループ）を取得し、
ブラウザ上でリアルタイム表示するテストツールです。

- **バックエンド**: Python + Flask + `moomoo-api`（旧 `futu-api` にも両対応）
- **リアルタイム配信**: Server-Sent Events (SSE)
- **フロントエンド**: 依存ライブラリなしの素の HTML/JS
- **モックモード**: OpenD 未接続でも擬似データで UI を試せます

---

## 構成

```
ブラウザ  ──SSE──▶  Flask (app.py)
                       │
                       ▼
              MoomooQuoteClient (moomoo_client.py)
                       │  TCP 127.0.0.1:11111
                       ▼
              moomoo OpenD ゲートウェイ ──▶ moomoo サーバー（実データ）
```

OpenD が起動していない場合は自動でモックモードに切り替わり、擬似的な
ランダムウォークのデータを表示します（UI 動作確認用）。

## セットアップ

### 1. Python 依存をインストール

```bash
pip install -r requirements.txt
```

### 2. 実データの取得手順（moomoo OpenD）

実際のリアルタイムデータを取得するには **moomoo証券の口座** と **OpenD ゲートウェイ** が必要です。
OpenD はローカル PC（Windows / macOS / Linux）上で動かします。

#### 2-1. OpenD のダウンロードと起動

1. [moomoo OpenAPI ページ](https://www.moomoo.com/download/OpenAPI) から
   **OpenD** をダウンロードします（GUI 版が簡単。Windows / macOS 推奨）。
2. OpenD を起動し、**moomoo ID（口座のログイン情報）でログイン**します。
   初回は SMS 等の二要素認証が求められます。
3. ログイン完了後、OpenD はデフォルトで `127.0.0.1:11111` で API を待ち受けます。

#### 2-2. 日本株リアルタイム相場の権限確認

- moomoo証券（日本）の口座があれば、**東証のリアルタイム相場（Level 1）は通常無料**で
  付与されています。moomoo アプリの「市場データ」画面で確認できます。
- 権限が無い場合、API の購読 (`subscribe`) がエラーになるか遅延データになります。

#### 2-3. 接続診断

Web 画面を起動する前に、診断ツールでどこまで通っているか確認できます:

```bash
python check_opend.py                # 9984 で診断
python check_opend.py --watch 10    # 10秒間リアルタイムプッシュを観測
```

チェック内容: OpenD ポート到達性 → SDK → OpenD ログイン状態 → 日本市場の状態 →
9984 の購読・クォート取得 → プッシュ受信。失敗した段階で原因と対処を表示します。

> OpenD を起動していない場合でも、Web ツール自体はモックモードで動作します。

### 3. 起動

```bash
python app.py                       # デフォルト: 9984
python app.py --codes 9984,7203,6758   # 複数銘柄
python app.py --codes US.GOOGL          # 米国株（市場プレフィックスを明示）
python app.py --host 127.0.0.1 --port 11111   # OpenD の接続先を指定
```

ブラウザで **http://127.0.0.1:5000** を開きます。

### 米国株・他市場のテスト

プレフィックス無しのコードは日本株 (`JP.`) として扱われます。
他市場は市場プレフィックスを付けて指定します:

| 市場 | 例 |
|------|-----|
| 日本株 | `9984` または `JP.9984` |
| 米国株 | `US.GOOGL`（Google/Alphabet）, `US.AAPL` |
| 香港株 | `HK.00700`（テンセント） |

日本株の相場権限が無い場合でも、米国株のリアルタイム（LV1）は
moomoo 口座で無料付与されていることが多いので、接続テストに便利です:

```bash
python check_opend.py --code US.GOOGL --watch 10   # 診断
python app.py --codes US.GOOGL                     # Web 表示
```

> 米国市場の取引時間は日本時間 22:30〜翌 5:00（夏時間）/ 23:30〜翌 6:00（冬時間）。
> 時間外はクォート値は取れますが、プッシュ更新はほぼ発生しません。

## コマンドラインオプション

| オプション | デフォルト | 説明 |
|-----------|-----------|------|
| `--codes` | `9984` | 銘柄コード（カンマ区切り）。`JP.` プレフィックスは省略可 |
| `--host` | `127.0.0.1` | OpenD のホスト |
| `--port` | `11111` | OpenD のポート |
| `--web-host` | `127.0.0.1` | Web サーバーのホスト |
| `--web-port` | `5000` | Web サーバーのポート |

## 画面の見方

- 右上のバッジが **REALTIME**（緑）なら実データ、**MOCK**（橙）なら擬似データです。
- 色は日本式（**上昇 = 赤 / 下落 = 緑**）で表示します。
- 価格が更新されるとカードが一瞬フラッシュします。
- 表示項目: 現在値・前日比・始値・前日終値・高値・安値・出来高・売買代金。

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `app.py` | Flask アプリ本体。SSE でリアルタイム配信 |
| `moomoo_client.py` | OpenD への接続・購読・モックデータ生成 |
| `check_opend.py` | OpenD 接続・実データ取得の診断ツール |
| `templates/index.html` | リアルタイム表示のフロントエンド |
| `requirements.txt` | Python 依存 |

## 注意事項

- 本ツールは **テスト / 学習目的** のものです。投資判断には利用しないでください。
- リアルタイム相場の取得には moomoo のアカウント・OpenD・相場データ権限が必要です。
- 日本株コードは内部で `JP.9984` の形式に正規化されます。
